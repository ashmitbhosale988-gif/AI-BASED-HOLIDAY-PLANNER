✨ Overview

🎯 AI-Based Holiday Planner is a smart travel planning system that generates personalized itineraries based on user preferences like budget, destination, and interests.

Instead of spending hours planning trips manually, this system uses Artificial Intelligence to create optimized travel plans instantly.

---

🚀 Features

✨ Smart itinerary generation
📍 Location-based recommendations
💰 Budget-friendly travel plans
🧠 AI-powered suggestions
📅 Day-wise planning
🌐 User-friendly interface

---

🛠️ Tech Stack

Frontend   : HTML, CSS, JavaScript
Backend    : Node.js / Express
AI API     : OpenAI API
Database   : MongoDB / Firebase
Deployment : Vercel

---

🧠 How It Works

1. User enters travel preferences
2. AI processes inputs (budget, days, interests)
3. Generates optimized itinerary
4. Displays complete travel plan

---

📊 System Architecture

User Input → AI Engine → Processing → Itinerary Output
